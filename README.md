# GST Calculator Pro

A modern, responsive, and user-friendly GST Calculator web application built using **HTML5, CSS3, Bootstrap 5, and Vanilla JavaScript**. This project helps users calculate GST, remove GST from inclusive prices, view CGST/SGST breakdowns, perform scientific calculations, and convert units—all in one place.

---

## 📌 Overview

GST Calculator Pro is designed for shopkeepers, business owners, accountants, freelancers, students, and anyone who needs quick and accurate GST calculations. The application works entirely in the browser without requiring any installation or backend server.

---

## ✨ Features

### GST Calculator

* Add GST to any amount
* Remove GST from GST-inclusive amount
* Supports 3%, 5%, 12%, 18%, 28%, and custom GST rates
* Automatic CGST and SGST calculation
* Instant real-time calculation

### Scientific Calculator

* Basic arithmetic operations
* Square root
* Percentage
* Power
* Trigonometric functions
* Logarithmic functions
* Memory functions (if implemented)

### Unit Converter

* Length conversion
* Weight conversion
* Temperature conversion
* Area conversion
* Volume conversion
* Time conversion
* Speed conversion

### User Experience

* Modern responsive interface
* Mobile-friendly design
* Dark/Light mode
* Calculation history
* Copy calculation results
* Export history to CSV
* Fast and lightweight
* No page reload required
